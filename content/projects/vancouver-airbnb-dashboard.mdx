Interactive dashboard for analyzing Vancouver Airbnb investment opportunities. Built with Python, Streamlit, and data visualization libraries to help investors make informed decisions about property investments.

## Key Features

- **Interactive Maps**: Visualize Airbnb listings across Vancouver neighborhoods
- **Investment Analysis**: Calculate potential ROI and revenue projections
- **Market Trends**: Analyze pricing patterns and seasonal variations
- **Property Comparison**: Compare different properties and neighborhoods
- **Data Visualization**: Charts and graphs for market insights

## Technologies Used

- **Python**: Data processing and analysis
- **Streamlit**: Interactive web dashboard
- **Pandas**: Data manipulation and analysis
- **Plotly**: Interactive visualizations
- **Folium**: Interactive maps
- **Scikit-learn**: Machine learning for price prediction

## Project Highlights

- **Real Estate Data Analysis**: Comprehensive analysis of Vancouver's Airbnb market
- **Investment Decision Support**: Tools to help investors evaluate properties
- **Interactive Visualizations**: Dynamic charts and maps for better insights
- **Market Intelligence**: Trends and patterns in the Vancouver rental market
