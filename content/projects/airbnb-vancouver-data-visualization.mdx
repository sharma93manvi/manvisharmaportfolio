Interactive Tableau dashboard analyzing Airbnb data in Vancouver. Features comprehensive data visualization including pricing trends, neighborhood analysis, and property insights for the Vancouver market.

## Key Features

- **Interactive Dashboard**: Comprehensive Tableau visualization of Vancouver Airbnb data
- **Pricing Analysis**: Trends and patterns in Airbnb pricing across neighborhoods
- **Neighborhood Insights**: Detailed analysis of different Vancouver areas
- **Property Types**: Analysis of various accommodation types and their performance
- **Market Trends**: Seasonal and temporal patterns in the Vancouver market

## Technologies Used

- **Tableau**: Advanced data visualization and dashboard creation
- **Data Analysis**: Statistical analysis of real estate data
- **Business Intelligence**: Converting data into actionable insights
- **Data Storytelling**: Presenting complex data in accessible formats
