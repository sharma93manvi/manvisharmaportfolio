AI-powered web app that generates stylish, Instagram-ready captions from photos. Built using Streamlit, OpenAI GPT-4o, and PostgreSQL for feedback tracking.

This project combines computer vision and natural language processing to create engaging social media content. The app analyzes uploaded images and generates contextually relevant, stylized captions that are perfect for Instagram posts.

**Key Features:**
- Image analysis and caption generation
- Multiple caption styles and tones
- User feedback tracking with PostgreSQL
- Modern web interface with Streamlit
- Integration with OpenAI's GPT-4o model

**Technologies Used:**
- Python
- Streamlit
- OpenAI GPT-4o
- PostgreSQL
- Computer Vision
- Natural Language Processing
