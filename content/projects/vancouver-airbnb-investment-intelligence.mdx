Interactive Vancouver Airbnb Investment Intelligence Dashboard with real market data, ROI analysis, competitive intelligence, and neighborhood insights. Built with Streamlit and powered by comprehensive market data.

## Key Features

- **Market Intelligence**: Real-time market overview with key metrics including average prices, ROI potential, and market opportunities
- **Investment Grading System**: Automated classification of properties (Excellent, Good, Fair, Poor)
- **Interactive Mapping**: Geographic visualization with property locations and key metrics
- **ROI Analysis**: Automated return on investment calculations and revenue forecasting
- **Competitive Intelligence**: Market positioning analysis and KMeans clustering for competitive grouping
- **Neighborhood Insights**: Comprehensive neighborhood-level performance metrics and rankings

## Technologies Used

- **Streamlit**: Interactive web dashboard framework
- **Python**: Data processing and analysis
- **Pandas & NumPy**: Data manipulation and numerical computing
- **Plotly & Folium**: Interactive visualizations and mapping
- **Scikit-learn**: Machine learning (KMeans clustering)
- **GeoPandas**: Geographic data analysis
- **Selenium & BeautifulSoup**: Web scraping capabilities

## Project Highlights

- **Real Market Data**: Uses comprehensive Vancouver Airbnb data from Inside Airbnb
- **Investment Analysis**: ROI calculations, revenue forecasting, and occupancy rate analysis
- **Market Saturation Analysis**: Identifies oversaturated vs. opportunity-rich areas
- **Growth Potential Scoring**: Predictive analysis for future market performance
- **Interactive Filters**: Neighborhood selection, property type, price range, and investment grade filtering
- **Live Demo**: Deployed Streamlit application with real-time data

## Dashboard Features

- **Interactive Maps**: Folium-based geographic visualization
- **Scatter Plots**: Market positioning and competitive analysis
- **Bar Charts**: Performance comparisons and rankings
- **Data Tables**: Detailed property information
- **KPI Cards**: Key performance indicators
- **Investment Grading**: Automated property classification system
